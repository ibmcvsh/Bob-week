## 📋 Prerequisites

Before diving into the labs, ensure you have the following tools installed:

| Tool | Description |
|------|-------------|
| 🐍 **Python 3.13+** | Required for MCP server operations |
| ⚡ **uv** | Fast Python package installer and resolver |
| 🐳 **Docker/Podman** | Container runtime for ODM deployment |
| 🔧 **Git** | Version control for repository management |

<details>
<summary>💡 <b>Installation Tips</b></summary>

- **Python**: Download from [python.org](https://www.python.org/downloads/)
- **uv**: Install via `pip install uv` or visit [astral.sh/uv](https://astral.sh/uv)
- **Docker**: Get Docker Desktop from [docker.com](https://www.docker.com/products/docker-desktop)
- **Rancher Desktop**: Alternative container runtime from [rancherdesktop.io](https://rancherdesktop.io/)
- **Git**: Download from [git-scm.com](https://git-scm.com/downloads)

</details>
