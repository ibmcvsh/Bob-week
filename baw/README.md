## 📋 Prerequisites

Before diving into the labs, ensure you have the following tools installed:

| Tool | Description |
|------|-------------|
| 🐍 **Python** | Required for automation scripts and testing |
| 📦 **Node.js** | Essential for widget development and tooling |
| ⚡ **uv** | Fast Python package installer and resolver |

<details>
<summary>💡 <b>Installation Tips</b></summary>

- **Python**: Download from [python.org](https://www.python.org/downloads/)
- **Node.js**: Get the LTS version from [nodejs.org](https://nodejs.org/)
- **uv**: Install via `pip install uv` or visit [astral.sh/uv](https://astral.sh/uv)

</details>
